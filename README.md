# Java Programs (MCA)

A collection of basic Java programs written as part of my MCA coursework and practice.  
These programs demonstrate core Java and object-oriented programming concepts.

---

## Programs

- **MatrixAddition.java**  
  Adds two matrices using 2D arrays.

- **Product.java**  
  Defines a `Product` class with basic attributes.

- **ProductList.java**  
  Demonstrates handling multiple `Product` objects.

---

## Concepts Covered

- Classes & Objects
- Arrays (1D & 2D)
- Basic OOP principles
- Control structures

---

## How to Run

```bash
javac MatrixAddition.java
java MatrixAddition
```
